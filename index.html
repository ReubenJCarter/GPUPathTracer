GPU Path Tracer

Intro

The point of this project is mostly educational. A few years ago, when I was still at uni, I watched a demo video for Brigade (Perhaps it was this). I was seriously impressed. I wanted to learn how systems such as octane and brigade actually work, this led me down a rather long and winding path which I am still on. For a day job I build systems to render billions of particles mostly from laser scan data which goes into TV shows,<>VR Experences<> and the like. Most of this work is based on quite traditional game engine style particle systems, essentially rasterising loads of little quads (lots of OpenGL and clever out of core data structures etc.) But not a lot of lighting! My aim with this project is to build a fast GPU based path tracing engine to learn how to simulate light and make photorealistic images really fast. I hope to document all of my findings here.
